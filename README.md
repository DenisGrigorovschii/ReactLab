# ReactLab - Каталог товаров

Лабораторная работа по React: CRUD приложение с использованием Context API и React Router.

## Описание

Приложение для управления каталогом товаров с полным функционалом CRUD (Create, Read, Update, Delete). Реализованы три лабораторные работы:

### Лабораторная работа №1
- Отображение списка сущностей с загрузкой через API
- Просмотр детальной информации в модальном окне
- Редактирование сущностей
- Создание новых сущностей
- Удаление сущностей

### Лабораторная работа №2
- Поиск по названию и описанию
- Фильтрация по категориям
- Фильтрация по диапазону цен
- Сортировка по цене и названию
- Оптимизация с использованием useMemo
- Состояния загрузки, ошибок и пустых результатов

### Лабораторная работа №3
- Context API для глобального управления состоянием
- Маршрутизация с React Router
- Страницы: список, детали, создание, редактирование
- Передача данных между страницами без props drilling

## Технологии

- React 18.2.0
- React Router DOM 6.14.1
- Vite 5.0.0
- Context API
- Hooks: useState, useEffect, useMemo, useContext, useParams, useNavigate

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

3. Откройте браузер по адресу, указанному в консоли (обычно http://localhost:5173)

## Структура проекта

```
src/
```

## API

Приложение использует [DummyJSON API](https://dummyjson.com/) для получения данных о товарах.
